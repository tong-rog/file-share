GitHub部署指南

1

创建GitHub仓库

登录GitHub → 点击右上角"+" → 选择"New repository" → 填写仓库名称 → 点击"Create repository"

2

上传文件

在仓库页面点击"Add file" → 选择"Upload files" → 上传index.html和files.json

3

启用GitHub Pages

仓库设置 → "Pages" → 选择main分支和根目录 → 保存

4

获取文件URL

点击files.json → 点击"Raw" → 复制浏览器地址栏URL → 替换下方代码中的JSON_FILE_URL

5

分享链接

GitHub Pages提供的链接格式为：https://<username>.github.io/<repository>
